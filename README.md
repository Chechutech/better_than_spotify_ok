# Music App: "Better than Spotify"

This a music-clone app made with "Spotify API" and built with Ionic and Nest js.

## Installation

Use the package manager [npm](https://pip.pypa.io/en/stable/) to install.

```bash
npm install
```

## ▶️ Run client

```bash
ionic serve
```
## ▶️ Run Server

```bash
npm run start
```
